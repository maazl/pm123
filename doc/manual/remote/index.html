PM123's Remote Command Reference

PM123 currently understands the following commands through the remote interface. The commands are grouped into the categories: general, playback control, playlists, playlist manipulation, user interface, meta data, configuration and plug-ins.

All commands return an empty line at the end. This is different from PM123 up to version 1.40. However, unless you send no more than one command per pipe instance it makes no difference.

Commands that start with a plug-in name: are plug-in specific commands and do only work if the specified plug-in is loaded.

Note that the commands must have an asterisk (*) as prefix if the are sent through the pipe. This is a tribute to compatibility with older PM123 versions.

Alphabetical list of all commands

  • add ... - add item to playlist
  • autouse - auto use playlist on add
  • cd - set current base URL or directory
  • clear - clear playlist
  • current - query current song or playlist
  • default ... - reset an option to its default value.
  • dir ... - add directory to playlist
  • float - float window on top
  • font - select font
  • forward - forward scan
  • getmessages - return descriptive (error) messages
  • hide ... - hide player or dialog window
  • info format - query file and format information
  • info invalidate - invalidate all meta information on the specified object
  • info meta - query meta information
  • info playlist - query information about a (nested) playlist
  • info refresh - refresh all meta information on the specified object
  • isvisible ... - query whether a dialog window is visible
  • jump ... - jump to location
  • load ... - load file/URL
  • navigate - query current location and optionally navigate within (nested) playlists
  • next - skip next
  • option ... - set/query configuration options
  • pause - pause
  • pl add ... - add item to playlist
  • pl callstack - query the call stack of the current item in a nested playlist tree
  • pl current - query current playlist
  • pl depth - query the call stack depth of the current item within the selected playlist
  • pl enter - enter a nested playlist
  • pl index - query current playlist item index
  • pl info format - query format information about the current playlist item
  • pl info item - query item information about the current playlist item
  • pl info meta - query meta information about the current playlist item
  • pl info playlist - query recursive playlist information about the current playlist item
  • pl item - query the current playlist item
  • pl itemindex - return the index of the selected song in a nested playlist
  • pl leave - leave a (nested) playlist
  • pl location - query current location within a nested playlist
  • pl navigate ... - navigate to item
  • pl next - next song in nested playlist tree
  • pl nextitem - next item in a playlist
  • pl parent - query lowermost playlist in the current call stack
  • pl prev - previous song in nested playlist tree
  • pl previtem - previous item in a playlist
  • pl remove - remove item from playlist
  • pl reset - reset playlist iterator
  • pl save - save playlist
  • pl setitem - modify the current playlist item
  • pl sort - sort playlist
  • play - play item
  • playlist - select playlist
  • playonload - start playing on load
  • plugin list ... - query or set the list of loaded plug-ins
  • plugin load ... - load a plug-in into the player
  • plugin params ... - query or set plug-in parameters
  • plugin unload ... - unload a plug-in
  • prev, previous - skip previous
  • query - query playback mode (like playing, auto repeat and so on)
  • quit - quit player
  • rdir ... - recursive add directory to playlist
  • remove - remove item from playlist
  • repeat - autorepeat mode
  • reset - reset the pipe interface to its initial state
  • rewind - reverse scan
  • save - save playlist
  • savestream - save stream as file
  • size - set player size
  • show ... - open dialog window
  • shuffle - enable shuffle
  • skin - load skin
  • status - get info string
  • stop - stop playback
  • time - current playing time
  • use - use playlist
  • version - query PM123 version
  • volume - volume control
  • write meta rst - modify meta information
  • write meta set - modify meta information
  • write meta to - modify meta information
